IMU 姿态估计系统模型

基于无迹卡尔曼滤波（UKF）的状态估计框架

🔧 过程模型 · 观测模型 · 噪声参数
📐

状态定义

State Vector xk ∈ ℝ7
xk = [ qk, ωk ]
前 4 维
qk ∈ ℝ⁴
机体相对世界坐标系的姿态（四元数）
后 3 维
ωk ∈ ℝ³
机体坐标系下的角速度 [Wx, Wy, Wz]
🔄

运动方程（Process Model）

xk → xk+1

描述状态如何从 k 时刻演化到 k+1 时刻：

qk+1 = qk ⊗ from_axis_angle(ωk · dt)
ωk+1 = ωk
# 对应代码（第 128–132 行） q_delta = Quaternion() q_delta.from_axis_angle(sw[i] * dt) # ω·dt = 这段时间转过的角度 qi_new = sq[i] * q_delta # q_{k+1} = q_k ⊗ q_Δ pw[i] = sw[i] # ω 不变
物理意义： ω·dt 表示在 dt 时间内机器人绕某轴转过的小角度（向量）； from_axis_angle 将其转换为旋转四元数 qΔ； qk ⊗ qΔ 将小旋转叠加到当前姿态上。

假设：两帧之间角速度恒定（匀速旋转模型），陀螺仪采样频率高（~100 Hz），此假设成立。
📡

观测方程（Measurement Model）

zk ∈ ℝ⁶

① 加速度计测量

静止时，测量重力在机体坐标系下的分量：

gworld = [0, 0, 9.81]
gbody = q−1 ⊗ [0, gworld] ⊗ q
# 对应代码（第 155 行） Z[:3, i] = world_to_body(sq2[i], g_world) # 即：(q⁻¹ × [0, 0, 9.81] × q).vec()

② 陀螺仪测量

直接测量角速度：

zgyro = ωk
# 对应代码（第 156 行） Z[3:, i] = sw2[i] # 直接等于角速度 sigma point

③ 完整观测方程

zk = [ q−1 ⊗ [0,0,9.81] ⊗ q, ωk ]T + ν
其中 ν ~ N(0, Qmeas)
# 噪声协方差（第 42–43 行） Q_meas = np.diag([1e1, 1e1, 1e1, # 加速度计方差（噪声大） 1e-1, 1e-1, 1e-1]) # 陀螺仪方差（噪声小）
🗺️

完整系统流程

预测 → 更新

运动方程 f(x)

q' = q ⊗ qΔ(ω·dt)
ω' = ω
+ 过程噪声 Rproc · dt
→

观测方程 h(x)

z[:3] = q−1·g·q ← 加速度计
z[3:] = ω ← 陀螺仪
+ 测量噪声 Qmeas
⚖️

两个方程的互补关系

运动方程 f 观测方程 h
输入 上一时刻状态 当前时刻状态
输出 预测下一时刻状态 预测传感器读数
信息来源 陀螺仪积分（ω·dt） 加速度计 + 陀螺仪
问题 积分漂移 加速度计噪声 · yaw 不可观测
优势 短时精确 长时稳定
💡

UKF 融合的核心直觉

陀螺仪短时精确但会积分漂移；加速度计有噪声，但提供绝对重力参考，长期稳定。UKF 将两者在最优意义下融合——在快速运动时信任陀螺仪，在静止或慢速时向加速度计校正，实现互补。
